1. **Reproduce:** Understand what's happening vs what should happen
2. **Locate:** Find the relevant code (check both client and server)
3. **Diagnose:** Identify the root cause — explain it briefly
4. **Fix:** Make the minimal change needed
5. **Verify:** Run `npm run check` to ensure no type errors
6. **Remember:** Save the lesson to memory so future sessions know about this

If the fix isn't working after 2-3 attempts, **STOP** and tell me what you've tried. Don't rabbit-hole.
